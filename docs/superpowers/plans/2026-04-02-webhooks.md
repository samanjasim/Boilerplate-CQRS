# Webhooks Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement tenant-scoped webhook endpoints with HMAC-SHA256 signed payloads, configurable event subscriptions, MassTransit delivery with exponential backoff retries, and full delivery audit log.

**Architecture:** Domain events on core entities (User, File, Role) fire MediatR handlers that publish `DeliverWebhookMessage` to RabbitMQ via MassTransit. A consumer finds matching webhook endpoints per tenant, delivers HTTP POST with HMAC signature, and records delivery status. Feature-flag gated per billing plan.

**Tech Stack:** .NET 10, EF Core, PostgreSQL, Redis, MassTransit/RabbitMQ, MediatR, React 19, TanStack Query, shadcn/ui

**Spec:** `docs/superpowers/specs/2026-04-02-webhooks-design.md`

- [ ] **Step 2:** Create `DeliverWebhookConsumer.cs`

Implements `IConsumer<DeliverWebhookMessage>`. Logic:
1. Load active WebhookEndpoints for `message.TenantId` subscribed to `message.EventType` (use `IgnoreQueryFilters`)
2. For each endpoint:
   a. Compute HMAC-SHA256 signature: `t={timestamp},v1={hmac(secret, "{timestamp}.{payload}")}`
   b. HTTP POST to endpoint URL with `Content-Type: application/json`, `X-Webhook-Signature-256` header
   c. Create `WebhookDelivery` record with Success or Failed status
   d. Cap response body to 4KB
3. Use `IHttpClientFactory` for HTTP calls
4. Try/catch per endpoint — one failure doesn't block others
5. Use `IServiceScopeFactory` to create scoped DbContext (consumer is singleton-ish in MassTransit)

**HMAC computation:**
```csharp
using System.Security.Cryptography;
using System.Text;

var timestamp = DateTimeOffset.UtcNow.ToUnixTimeSeconds();
var signedPayload = $"{timestamp}.{payload}";
using var hmac = new HMACSHA256(Encoding.UTF8.GetBytes(endpoint.Secret));
var hash = hmac.ComputeHash(Encoding.UTF8.GetBytes(signedPayload));
var signature = $"t={timestamp},v1={Convert.ToHexStringLower(hash)}";
```

```csharp
internal sealed class WebhookUserEventHandler(
    IApplicationDbContext context,
    IMessagePublisher messagePublisher,
    ILogger<WebhookUserEventHandler> logger) : INotificationHandler<UserCreatedEvent>
{
    public async Task Handle(UserCreatedEvent notification, CancellationToken cancellationToken)
    {
        if (notification.TenantId is null) return; // Skip platform-level events

        // Check if any endpoints are subscribed to this event type
        var hasSubscribers = await context.WebhookEndpoints
            .IgnoreQueryFilters()
            .AnyAsync(e => e.TenantId == notification.TenantId && e.IsActive, cancellationToken);

        if (!hasSubscribers) return;

        var payload = JsonSerializer.Serialize(new
        {
            id = $"evt_{Guid.NewGuid():N}",
            type = "user.created",
            tenantId = notification.TenantId,
            timestamp = DateTime.UtcNow,
            data = new
            {
                userId = notification.UserId,
                email = notification.Email,
                firstName = notification.FirstName,
                lastName = notification.LastName,
            }
        });

        await messagePublisher.PublishAsync(new DeliverWebhookMessage(
            notification.TenantId.Value, "user.created", payload, DateTime.UtcNow), cancellationToken);
    }
}
```

Create similar handlers for: UserUpdated, FileUploaded, FileDeleted, RoleCreated, RoleUpdated, InvitationAccepted (from AcceptInviteCommandHandler), SubscriptionChanged (existing event).

- [ ] **Step 1:** Create `WebhookDeliveryCleanupJob.cs`

Follow the `OrphanFileCleanupService` pattern. Runs daily. Deletes WebhookDelivery records older than 7 days (configurable via system setting `webhooks.delivery_retention_days`).

```csharp
public sealed class WebhookDeliveryCleanupJob(
    IServiceScopeFactory scopeFactory,
    ILogger<WebhookDeliveryCleanupJob> logger) : BackgroundService
{
    protected override async Task ExecuteAsync(CancellationToken stoppingToken)
    {
        while (!stoppingToken.IsCancellationRequested)
        {
            try
            {
                using var scope = scopeFactory.CreateScope();
                var context = scope.ServiceProvider.GetRequiredService<ApplicationDbContext>();
                var retentionDays = 7;

                var cutoff = DateTime.UtcNow.AddDays(-retentionDays);
                var deleted = await context.WebhookDeliveries
                    .IgnoreQueryFilters()
                    .Where(d => d.CreatedAt < cutoff)
                    .ExecuteDeleteAsync(stoppingToken);

                if (deleted > 0)
                    logger.LogInformation("Cleaned up {Count} webhook deliveries older than {Days} days", deleted, retentionDays);
            }
            catch (Exception ex)
            {
                logger.LogError(ex, "Error during webhook delivery cleanup");
            }

            await Task.Delay(TimeSpan.FromHours(24), stoppingToken);
        }
    }
}
```
